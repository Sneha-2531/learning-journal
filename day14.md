# String Questions
Write a program to input a string and print it.

Write a program to find the length of a string.

Write a program to convert a string to uppercase.

Write a program to convert a string to lowercase.

Write a program to print the first character of a string.

Write a program to print the last character of a string.

Write a program to count the number of characters in a string (excluding spaces).

Write a program to check whether a given character exists in a string.

Write a program to concatenate (join) two strings.

# Concatenation Questions

Input a city name and state name, then print: City, State.

Input a product name and price, then print them using concatenation.

Input a student's name and roll number, then display both in one sentence.

# Formatting Questions

Input two numbers and print their sum using an f-string.

Input a book name and author name, then display them using .format().

Input a company's name and location, then print a formatted sentence.

Input length and width, then print the area using an f-string.
