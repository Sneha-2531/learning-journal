# Day 18 – Cloud Deployment Models

**Date:** 2026-06-19  

## Models Learned
- Public Cloud → Shared, cost-efficient, scalable.  
- Private Cloud → Secure, controlled, expensive.  
- Hybrid Cloud → Mix of public + private.  
- Community Cloud → Shared by similar organizations.  
- Multi-cloud → Multiple providers for resilience.  

## Reflection
- Easy: Public vs Private distinction.  
- Challenging: Hybrid vs Multi-cloud differences.
