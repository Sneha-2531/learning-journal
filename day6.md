# Day 6 – Microsoft Excel Advanced Worksheet Management

**Date:** 2026-06-07  

## Topics Learned

### 1. Data Relocation Techniques
- Moving data using drag and drop  
- Cut data using `Ctrl + X`  
- Paste data using `Ctrl + V`  
- Relocating data efficiently within a worksheet  

### 2. Row and Column Management
- Inserting new rows using `Ctrl + Shift + +`  
- Adding rows through the right-click **Insert** option  
- Inserting and managing columns  
- Removing rows and columns when required  

### 3. Adjusting Cell Dimensions
- Changing row height and column width by dragging  
- Using double-click to auto-fit rows and columns  
- Adjusting multiple columns at once  
- Improving worksheet readability through proper sizing  

### 4. Hiding and Unhiding Rows and Columns
- Hiding rows and columns using the right-click menu  
- Unhiding hidden rows and columns  
- Managing worksheet visibility for better organization  

### 5. Worksheet Renaming
- Renaming sheets by double-clicking the sheet tab  
- Renaming sheets using the right-click **Rename** option  
- Organizing worksheets with meaningful names  

### 6. Worksheet Deletion
- Deleting unnecessary worksheets  
- Understanding that unsaved deleted sheets may not be recoverable  
- Managing workbook structure efficiently  

### 7. Worksheet Relocation and Duplication
- Moving worksheets by dragging sheet tabs  
- Copying worksheets using `Ctrl + Drag`  
- Using **Move or Copy Sheet** option  
- Creating copies of worksheets in the same or a new workbook  

## Skills Practiced
- Worksheet organization  
- Data management  
- Row and column operations  
- Sheet customization  
- Workbook maintenance
