# Day 7 – Python String Manipulation

**Date:** 2026-06-08  

## Topics Learned

### 1. String Concatenation
 Combining two or more strings using the `+` operator
-python
first_name = "Sneha"
last_name = "BS"
full_name = first_name + " " + last_name
print(full_name)

2. String Repetition
Repeating a string multiple times using the * operator
code:
message = "Warning! "
print(message * 3)

3. Common String Methods
upper() – Converts text to uppercase

lower() – Converts text to lowercase

strip() – Removes leading and trailing spaces

replace(old, new) – Replaces part of a string with another string

len() – Returns the length of a string

4. Quotes in Strings
Using single quotes ' ' and double quotes " "

Including quotes inside strings

5. String Indexing
Accessing individual characters using index positions
python:
name = "Ganesha"
print(name[0])   # First character

6. String Slicing
Extracting a part of a string
python:
name = "Ganesha"
print(name[0:4])

7. Negative Indexing
Accessing characters from the end of a string
python:
print(name[-1])

8. Advanced Slicing
Using [start:end:step] to skip characters
python:
print(name[::2])

9. Escape Sequences
\n → New line

\t → Tab space
python:
print("Sneha\nis a good girl")
print("Sneha\tis a good girl")

Key Takeaway
Today I gained a solid understanding of Python strings, including concatenation, repetition, common methods, indexing, slicing, negative indexing, advanced slicing, and escape sequences. These concepts are essential for handling and manipulating text data in Python programs.
