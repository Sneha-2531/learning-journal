# Day 5 – Excel Functions and Formulas

**Date:** 2026-06-06  

## Topics Covered
- **Introduction to Excel Functions**  
  Functions are predefined formulas used to perform calculations.  
  Structure: `=FunctionName(Arguments)`  

- **SUM Function**  
  Adds values in a range.  
  Example: `=SUM(B4:B8)`  

- **MIN and MAX Functions**  
  `MIN()` finds the smallest value.  
  `MAX()` finds the largest value.  

- **AVERAGE Function**  
  Calculates the average of selected numbers.  
  Example: `=AVERAGE(A1:A10)`  

- **COUNT Function**  
  Counts cells containing numeric values.  
  Example: `=COUNT(A1:A10)`  

- **Function Arguments Window**  
  Used the Function Wizard (fx) to search and insert functions.  

- **AutoSum Feature**  
  Quickly calculates totals.  
  Shortcut: `Alt + =`  

- **AutoFill for Formulas**  
  Copied formulas to adjacent cells.  
  Learned relative vs. absolute references.  

- **SUMPRODUCT Function**  
  Multiplies corresponding values in arrays and adds results.  
  Useful for total revenue (price × quantity).  

- **Error Handling in Excel**  
  Identified and ignored certain formula-related errors.  

## Key Takeaway
Hands-on practice with essential Excel functions — **SUM, MIN, MAX, AVERAGE, COUNT, AutoSum, AutoFill, SUMPRODUCT** — all of which are powerful tools for efficient calculations and data analysis.
