# Day 16 – Python Operators Practice

**Date:** 2026-06-16  

## Topics Covered
- Assignment Operators  
- Comparison Operators  
- Logical Operators  
- Membership Operators  

## Exercises

### Assignment Operators
1. Input a number and add 10 to it using `+=`.  
2. Input a number and subtract 5 from it using `-=`.  
3. Input a number and multiply it by 3 using `*=`.  
4. Input a number and divide it by 2 using `/=`.  

### Comparison Operators
5. Input two numbers and check whether they are equal (`==`).  
6. Input two numbers and check whether the first number is greater than the second (`>`).  
7. Input two numbers and check whether the first number is less than or equal to the second (`<=`).  
8. Input two numbers and check whether they are not equal (`!=`).  

### Logical Operators
9. Input your age and check if you are between 18 and 60 using `and`.  
10. Input a number and check if it is less than 0 or greater than 100 using `or`.  
11. Input a boolean value (`True` or `False`) and print its opposite using `not`.  
12. Input marks and attendance percentage. Check if marks are above 40 and attendance is above 75%.  

### Membership Operators
13. Input a string and a character. Check whether the character is present in the string using `in`.  
14. Input a list of fruits and check whether `"apple"` is present in the list.  
15. Input a string and check whether the letter `"z"` is not present using `not in`.  

## Key Takeaway
Today, I practiced **assignment, comparison, logical, and membership operators** in Python. These operators help in updating values, comparing data, making logical decisions, and checking membership in sequences.
