# Day 13 – Excel Charts & Visualization  
 Date: June 14, 2026  

## Topics Covered
1. **Creating an Excel Column Chart**  
   - Select required data  
   - Insert → Charts → Column Chart  
   - Visualize data in column form  

2. **Using Excel Chart Ribbon for Precision**  
   - Add and customize chart elements  
   - Modify chart titles, legends, and labels  
   - Improve chart appearance with formatting tools  

3. **Importing & Enhancing Data in Excel Charts**  
   - Insert charts based on existing data  
   - Add additional data series when needed  
   - Adjust chart settings for better presentation  

4. **Applying Trendline Formatting Techniques**  
   - Add a Trendline to charts  
   - Format trendline style and appearance  
   - Use trendlines to analyze patterns and trends  

5. **Working with Pie Charts**  
   - Create pie charts from selected data  
   - Represent proportions and percentages visually  
   - Compare parts of a whole effectively  

6. **Design Tab & Chart Customization**  
   - Use Design tab for layouts and styles  
   - Change colors, chart styles, and appearance  
   - Improve readability and presentation quality  

## Key Learning Outcome
Gained hands-on experience in creating, formatting, customizing, and analyzing Excel charts, including Column Charts, Pie Charts, Trendlines, and Chart Design features for effective data visualization.
