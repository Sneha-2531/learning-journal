# Day 12 – Excel SmartArt & Communication  
 Date: June 13, 2026  

## Topics Covered
- Creating SmartArt graphics for visual communication  
- Inserting SmartArt using:  
  - Insert Tab  
  - Pictures  
  - Online Pictures  
- Converting pictures into SmartArt  
- Using SmartArt to represent processes and workflows  
- Customizing SmartArt shapes and layouts  
- Enhancing flow charts with predefined formatting  
- Understanding communication flow:  
  - Sender → Message → Receiver → Feedback  
- Designing process diagrams to show step-by-step information flow  
- Creating visual representations for reports and presentations  
- Learning how SmartArt improves clarity and makes information easier to understand  

## Key Learning Outcome
I learned how to use SmartArt in Excel to create professional diagrams, process flows, and communication models. SmartArt helps present information visually, making data more attractive, organized, and easy to understand.
