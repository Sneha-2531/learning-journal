# Day 10 – June 11, 2026  
**Topic: Python Lists**

## Topics Covered
- Creating and printing lists  
- Accessing list elements (positive & negative indexing)  
- Modifying list elements  
- Adding elements: `append()`, `insert()`  
- Removing elements: `remove()`, `pop()`, `clear()`  
- List slicing: `list[start:stop:step]`  
- Common list methods: `index()`, `count()`, `reverse()`, `sort()`  
- Built-in functions: `len()`, `sum()`  
- Working with nested lists  
- Accessing elements from nested lists  

## Key Learnings
- Lists allow easy storage and management of multiple items.  
- Elements can be added, modified, removed, and accessed efficiently.  
- Slicing helps retrieve specific portions of a list.  
- Built-in methods simplify common list operations.  
- Nested lists organize related data in a structured format.  

## Outcome
Gained practical experience in creating, manipulating, and analyzing lists using various Python list methods and functions. This knowledge is useful for handling collections of data in Python programs.
