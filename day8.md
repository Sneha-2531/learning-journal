# Day 8 – Python Operators

**Date:** 2026-06-09  

## Topics Covered
- None Data Type  
- Assignment Operators  
- Comparison Operators  
- Logical Operators  
- Membership Operators  
- Bitwise Operators  

## Key Learnings
### None Data Type
- Learned about the `None` value in Python.  
- Used the `type()` function to identify its data type.  

### Assignment Operators
- Learned how to assign values using `=`.  
- Practiced shorthand operators like `+=`.  

### Comparison Operators
- Compared values using:  
  - `==` (equal to)  
  - `!=` (not equal to)  
  - `>` (greater than)  
  - `<` (less than)  

### Logical Operators
- Learned the use of `and`, `or`, `not`.  
- Practiced combining and evaluating conditions.  

### Membership Operators
- Checked if a value exists in a string using `in` and `not in`.  

### Bitwise Operators
- Introduction to bitwise operations.  
- Learned binary representation of numbers.  
- Understood that 8 bits = 1 byte.  

## Key Takeaway
Today, I gained a better understanding of Python operators and how they help in performing assignments, comparisons, logical decisions, membership checks, and binary operations.
