## Topic Learned
- Merge Sort

## Key Points
- Merge sort is a divide-and-conquer algorithm.
- It splits the array into halves, recursively sorts them, and merges the sorted halves.
- Time Complexity: O(n log n)
- Space Complexity: O(n)

## Reflection
Today I understood how merge sort works step by step. Practicing with examples helped me see how the merging process builds the sorted array.
